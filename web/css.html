CSS alapok

Mi a CSS


A CSS a weboldalak megjelenítésének leírására szolgáló nyelv, beleértve a színeket, az elrendezést és a betűtípusokat.


Mire jó?


Lehetővé teszi a prezentáció adaptálását különböző típusú eszközökhöz, például nagy képernyőkre, kis képernyőkre vagy nyomtatókra.

CSS syntax.

A CSS egy szabályalapú nyelv - Ön határozza meg azokat a stíluscsoportokat meghatározó szabályokat, amelyeket a weboldal adott elemeire vagy elemcsoportjaira alkalmazni kell.
Például: "Azt szeretném, ha az oldalam főcíme nagy piros szövegként jelenjen meg."

A következő kód egy nagyon egyszerű CSS-szabályt mutat be, amely eléri a fent leírt stílust: 


h1 {
    color: red;
    font-size: 5em;
}

Hogyan is lehet használni?

A szabály egy választóval nyílik meg.
Ez kiválasztja azt a HTML-elemet, amelyet stílusozni fogunk.
Ezután van egy göndör kapcsos zárójelkészletünk { }.
Ezeken belül egy vagy több nyilatkozat lesz, amelyek tulajdon és érték párok formájában jelennek meg.
Mindegyik pár megadja a kiválasztott elem(ek) egy tulajdonságát, majd azt az értéket, amelyet a tulajdonságnak meg szeretnénk adni.

A kettőspont előtt van a tulajdonságunk, a kettőspont után pedig az érték.
A CSS-tulajdonságok különböző megengedett értékekkel rendelkeznek, attól függően, hogy melyik tulajdonság van megadva.
Példánkban rendelkezünk a color tulajdonsággal, amely különféle színértékeket vehet fel.
Megvan a font-size tulajdonság is.
Ez a tulajdonság különböző méretű egységeket vehet fel értékként.
Egy CSS-stíluslap sok ilyen szabályt tartalmaz, egymás után írva.Ebben az esetben az első szintű címsorokat stílusozzuk .
